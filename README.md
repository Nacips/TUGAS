# Dokumentasi Tugas Praktik PABP

Repositori ini berisi kumpulan latihan dan tugas kuliah Praktik Pemrograman Aplikasi Perangkat Bergerak (PABP).

## Pembagian Folder Aplikasi

* **Folder Android-studio**: Berisi semua kumpulan latihan dan tugas yang dibuat menggunakan **Android Studio**.
* **Folder React-Native**: Berisi semua kumpulan latihan dan tugas yang dibuat menggunakan **React Native**.
